# Audio asset licensing

## `sfx/*.wav` — project-original, no third-party license

Every file under `sfx/` is generated by `tools/gen_audio.py` (committed in
this repo, pure Python stdlib, deterministic). They are original output of
this project's own code — no third-party samples, packs, or recordings were
used — so they carry the project's own rights and no external attribution or
license obligations.

Regenerate (byte-identical) with:

    python tools/gen_audio.py

## `music/` — empty by design

Looping music is a taste call and is not sourced yet. When tracks are added:

1. Put the files here (`.ogg` preferred for web size).
2. Register them in `Audio.TRACKS` (`scripts/audio.gd`).
3. **Add their license + attribution to this file in the same commit** —
   `tools/check_audio.py` fails CI if an audio file lands without a LICENSES
   entry. CC0 sources that fit the project: kenney.nl (Music packs),
   opengameart.org (filter CC0) — both unreachable from the cloud build
   sandbox, so this is a local-machine step.
