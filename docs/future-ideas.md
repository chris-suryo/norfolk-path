## Arena / survival mode — roguelite spinoff  *(v3+, possible SEPARATE project)*

**Status:** idea only — not scoped, not started, explicitly gated on
norfolk-path v1 shipping first.

A separate **arena / survival mode** with **procedurally generated levels each
run** — a different genre from norfolk-path's fixed narrative quest (more of a
**roguelite**). It would reuse the **same assets and engine knowledge** (Cute
Fantasy pack, Godot 4.7, the combat/movement systems we've built) but is its own
thing: replayable, generated arenas rather than one hand-built story level.

Chris's intent: a place to reuse what we've learned once the narrative game is
done. Likely a **separate project/repo**, not a mode bolted onto norfolk-path.

When v1 has shipped and this is picked up, it deserves its own brief +
scope pass from scratch (procedural generation, run structure, meta-progression
are all real design work not addressed here).
